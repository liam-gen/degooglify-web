# degooglify-web
Système d'analyse pour dégoogliser un site web.

Exemple :
- Le système analyse un site et trouve Google Analytics. Il alterte l'utilisateur et propose des alternatives

- Google Analytics
  - Matomo
  - Plausible
- Google reCaptcha
  - hCaptcha
  - Cloudflare Turnstile
- Google Ads
  - Carbon ads
  - EthicalAds
- Google Fonts
  - fontsquirrel
- Google Maps
  - Open Street Map
